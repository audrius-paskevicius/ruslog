# What is ruslog?

logging library that extend the logrus

# Features

- [x] Manage multiple loggers. 
- [x] Manage of an Appender and Formatter. 
- [ ] File appender 
- [ ] Asynchronous log output


# Requirements

- [logrus](https://github.com/sirupsen/logrus)

# Getting started

```sh
$ go get github.com/dogenzaka/ruslog
```
